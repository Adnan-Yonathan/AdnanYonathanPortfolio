# Adnan Yonathan - Portfolio

A modern, professional landing page showcasing entrepreneurial ventures, app development projects, and expertise.

## Features

- **Modern Design**: Clean, professional aesthetic with smooth animations and micro-interactions
- **Responsive**: Fully mobile-responsive design that works on all devices
- **Performance Optimized**: Lightweight with no external dependencies (except Google Fonts)
- **Interactive Elements**: Smooth scroll, parallax effects, animated counters, and card hover effects

## Sections

- **Hero**: Eye-catching introduction with animated stats
- **About**: Personal story and skill highlights
- **Projects**: Featured projects including Delta AI, No-Code Academy, and SaaS portfolio
- **Ventures**: Overview of business initiatives
- **Contact**: Call-to-action with social links

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript (No frameworks required)

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Or deploy to any static hosting service (Vercel, Netlify, GitHub Pages)

## File Structure

```
├── index.html      # Main HTML file
├── styles.css      # All styles and animations
├── script.js       # Interactive functionality
└── README.md       # This file
```

## Customization

- **Colors**: Edit CSS variables in `:root` selector in `styles.css`
- **Content**: Update text in `index.html`
- **Social Links**: Replace placeholder URLs in the contact section
